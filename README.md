# ThelerineVTON

The deep learning virtual try-on engine powering ThelerineAI.

## Overview

ThelerineVTON is a geometry-aware virtual try-on framework designed to transfer garments onto people while preserving identity, pose, and realism. The project focuses on developing virtual try-on models that perform well on diverse African body types and fashion.

## Features

- Identity-preserving virtual try-on
- Multi-scale feature fusion
- DensePose-guided garment transfer
- Modular PyTorch architecture
- Production-ready training pipeline
